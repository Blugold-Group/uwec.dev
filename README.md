# uwec.dev

This is the code for [uwec.dev](https://uwec.dev), the unofficial student-run guide to the computer science department of [University of Wisconsin - Eau Claire](https://uwec.edu)

If you're a student and you want to contribute code, go ahead! Submit a PR and it'll probably be approved without an issue  

> This website is run solely by students without endorsement or input from any CS faculty.

Inspirations:
- [https://wiki.cs.jmu.edu/](https://wiki.cs.jmu.edu/)
- [https://wiki.cse.lsu.edu/](https://wiki.cse.lsu.edu/)
- [https://uwindsorcss.github.io/wiki/](https://uwindsorcss.github.io/wiki/)
